Datasets & Scores Dataset and score definitions

Datasets

STRING protein-protein interaction
Live Homo sapiens STRING API interaction partners. The confidence window filters STRING's combined interaction score. Only physical score requests direct physical PPI evidence; otherwise STRING can include broader functional and predictive associations.
DepMap CRISPR gene effect
Local CRISPRGeneEffect matrix. Mean Chronos is the average gene dependency score across cell lines; lower means stronger dependency. Essential proteins are genes with Chronos/CERES <= -1.0 in >=50% of scored cell lines.
IPDB, BindingDB, literature pockets
Local IPDB snapshot plus IPDB exports from the downloaded colleague project. BindingDB gives each candidate's best binder; literature_pockets.tsv adds curated pocket labels and ligandability scores.

Score glossary

STRING confidence
STRING combined edge confidence, 0-1. Channel scores such as experiments, databases, text mining, coexpression, neighborhood, fusion, and cooccurrence are per-evidence STRING subscores.
Mean Chronos
Mean DepMap CRISPR gene-effect score across cell lines. More negative values indicate stronger dependency.
BindingDB nM
Best IPDB BindingDB affinity for the protein. Lower nM values indicate stronger measured binding.
RIPTAC scores
Target cancer scores and effector-to-target pair scores come from the colleague-defined IPDB scorer; exact formula TBD by IPDB scorer.
Literature pocket ligandability
Curated pocket-level ligandability from literature_pockets.tsv. Exact scoring rubric TBD by IPDB scorer.
